Instrucciones:

Este componente trata de intentar dibujar a Baulbasour de la misma manera que sale en la imagen, para esto debes seleccionar un color y pinchar en un cuadro de la pizarra. Para empezar de nuevo solo debes seleccionar el boton Limpiar.

Notas:

- Esta funcionalidad consta de dos componentes, el componente hijo equivale a 1 pixel de la pizarra en donde se dibuja, mientras que el componente padre, es la compisicion de todos los pixeles necesarios para crear la pantalla, es decir 10x12 = 120 componentes hijos los cuales fueron reutilizados.

- Se planea implementar un sistema de tiempo para esta funcionalidad, esto es, que se tome el tiempo que demoran los usuarios al terminar el dibujo, para asi, tener un ranking top 5 de usuarios que demoraron menos tiempo terminar la tarea.
